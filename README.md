# Rajneesh Chauhan — Data Analyst Portfolio

A fully interactive, single-file portfolio website built from resume data.

## 🚀 Live Features

- **Animated particle network** — canvas-based floating dots that connect to your cursor
- **Custom cursor** — glowing dot + lagging ring with hover morphing
- **Typing animation** — cycles through specializations in the hero
- **Animated stat counters** — numbers count up on scroll (500K+, 89%, 150+, 22%)
- **3D tilt cards** — project cards tilt toward mouse on hover
- **Mouse glow** — radial highlight follows cursor inside skill cards
- **Scroll reveal** — staggered fade-in animations throughout
- **Active nav highlighting** — nav links highlight based on scroll position
- **Embedded resume** — download and view PDF directly from the page

## 🌐 How to Deploy for Free

### Option 1 — GitHub Pages (Recommended, easiest)

1. Create a free account at [github.com](https://github.com)
2. Create a new repository named `your-username.github.io` (e.g. `Codex610.github.io`)
3. Upload `portfolio.html` and rename it to `index.html`
4. Go to **Settings → Pages → Source → main branch → / (root)**
5. Save — your site will be live at `https://Codex610.github.io` within 1–2 minutes

> ✅ Free forever, custom domain support, HTTPS included

---

### Option 2 — Netlify (Drag & drop, zero config)

1. Go to [netlify.com](https://netlify.com) and sign up free
2. Drag and drop `portfolio.html` onto the Netlify dashboard
3. Rename it to `index.html` before uploading
4. Your site is instantly live at a `*.netlify.app` URL

> ✅ Free tier is very generous, one-click deploys

---

### Option 3 — Vercel

1. Go to [vercel.com](https://vercel.com) and sign up with GitHub
2. Create a new project, upload `index.html`
3. Deploy — live in seconds at `*.vercel.app`

> ✅ Fast CDN, free HTTPS

---

### Option 4 — Cloudflare Pages

1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Connect GitHub or upload files directly
3. Deploy — live at `*.pages.dev`

> ✅ Unlimited bandwidth on free tier
